# Back-End

Back-End voltado ao projeto da Fetin 2016 onde segue a ideia de desenvolver uma white label
completa onde ira ser extremamente modular seguindo sistemas hierárquicos onde teremos todos os usuários
executando tarefares disponibilizadas internamente seguindo a hierarquia do contratante.

Estamos utilizando DDD (Domain-Driven Design).
